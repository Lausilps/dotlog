# Visão do Produto

O **DotLog** é um aplicativo oficial da **CoreDot** criado para registrar pensamentos, memórias, ideias e projetos de forma rápida, organizada e segura.

O aplicativo nasceu da necessidade de preservar conteúdos pessoais que antes eram armazenados em conversas privadas do WhatsApp e que foram perdidos durante uma troca de celular.

Seu principal objetivo é transformar qualquer pensamento em um registro permanente com a menor quantidade possível de interações.

---

# Conceito

O DotLog **não é**:

* um diário tradicional;
* um bloco de notas comum;
* um gerenciador de tarefas.

O DotLog é um espaço onde qualquer pensamento pode ser registrado instantaneamente através de:

* Texto
* Áudio
* Foto
* Imagem

Mais do que armazenar arquivos, o DotLog registra momentos da vida e da criatividade.

Cada projeto representa uma história.

Cada registro representa um momento.

---

# Filosofia

> Registrar uma ideia deve ser tão rápido quanto enviar uma mensagem para si mesmo no WhatsApp.

---

# Regra Principal de UX

> **Nenhum registro deve exigir mais de dois toques para começar.**

Toda decisão de interface deverá seguir esse princípio.

---

# Estrutura do Aplicativo

O DotLog será dividido em dois conceitos principais.

---

# 1. Projetos

Os projetos funcionam como grupos do WhatsApp.

Exemplos:

* 💡 Pensamentos
* 📖 Meu Livro
* 💛 CoreDot
* 📷 Momentos
* 🚀 Ideias
* 🎓 Faculdade

Cada projeto possui sua própria linha do tempo.

Todas as configurações são realizadas apenas uma vez, no momento da criação.

## Configurações do Projeto

* Nome
* Descrição (opcional)
* Cor
* Ícone ou imagem
* Projeto favorito
* Privacidade
* Participantes (caso compartilhado)

---

# 2. Registros

Os registros funcionam como mensagens enviadas dentro de cada projeto.

Cada registro poderá conter:

* Texto
* Áudio gravado
* Áudio enviado
* Foto pela câmera
* Imagem da galeria

Funcionalidades futuras:

* Vídeos
* Documentos
* Links

Os registros **não possuem título obrigatório**.

O sistema identificará automaticamente cada registro utilizando a data e uma pequena prévia do conteúdo.

# Colaboração

O DotLog será preparado desde o início para suportar projetos compartilhados.

Existirão dois tipos de projeto.

## Projeto Pessoal

Apenas o proprietário possui acesso.

## Projeto Compartilhado

O proprietário poderá convidar colaboradores.

Todos os participantes poderão enviar:

* Texto
* Áudio
* Fotos
* Imagens

Cada registro exibirá automaticamente seu autor.

Exemplo:

**Laura**

🎙️ Áudio

**Fulano**

📝 Texto

**Laura**

📷 Imagem

---

# Permissões

## Proprietário

Pode:

* Editar projeto
* Excluir projeto
* Convidar colaboradores
* Remover colaboradores
* Exportar dados

## Colaborador

Pode:

* Visualizar projeto
* Criar registros
* Editar seus próprios registros
* Excluir seus próprios registros

Não poderá excluir o projeto nem apagar registros de outros participantes.

---

# Sincronização

O MVP utilizará sincronização simples entre dispositivos.

No futuro, poderá evoluir para sincronização em tempo real semelhante ao WhatsApp.
